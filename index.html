Overview:

Espresso is: "... a simple API for writing reliable UI tests ..."

This app show the usage of the testing library "espresso", which is now part of the Android Support Repository.

Code:

A test is just as simple as ...

public void testSimpleClickAndCheckText() {
        onView(withId(R.id.button_simple))
                .perform(click());

        onView(withId(R.id.text_simple))
                .check(matches(withText("Hello Espresso!")));
    }

and the Setup for the this test was just ...

 @Override
    public void setUp() throws Exception {
        super.setUp();
        // Espresso will not launch our activity for us, we must launch it via getActivity().
        getActivity();
    }
